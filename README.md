## Как запускать?

- Убедиться что установлен python версии выше 3.11
- Установить необходимые библиотеки если нет *(json, datetime, nextcord)*
- Зайди в папку /root 
- Запустить main.py 
> [!NOTE]
> Запуск на **Linux**: `python3 main.py`
> Запуск на **Windows**: `python main.py`
